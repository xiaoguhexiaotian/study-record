### 用于学习 vue-class-component 和 vue-property-decorator

vue-property-decorator 实质上是 vue-class-component 的超集

1. 语法上的不同，更接近 class 的写法
2. 更好的支持 ts 语法
3. 本质上与 vue2 没有太大区别，在运行时都是会构建为 vue2 代码
